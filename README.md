# Mit AR weniger Rückenschmerzen

## Idee

Mit Hilfe von AR (Augmented Reality) soll die Erfassung, Vermessung und Präsentation von archäologischen Ausgrabungen beschleunigt und verbessert werden.

## Vorbereitung

* Konto auf [arkio.is](https://arkio.is) anlegen
* Arkio App auf Meta Quest 2/3 und anderen Geräten installieren.

## Erfassung / Vermessung

### Karte / Messergebnisse auf Gelände plazieren

Bestehende Karten mit Grundrissen oder Messergebnissen können in Arkio geladen und auf dem Ausgrabungsgelände plaziert werden.
1. Zuerst muss dafür eine Gruppe für den Austausch der Daten erstellt werden: [Arkio Cloud Gruppen](https://cloud.arkio.is)
Dort dann die Dateien hochladen.
2. In der VR-Brille Create -> Imports klicken. Der Gruppe aus Schritt 1 beitreten. Dann die gewünschte Karte auswählen. Den Massstab passend zur Karte wählen.
3. Vor Ort in der AR-Darstellung die Karte passend zur Umgebung ausrichten indem Landmarken auf der Karte mit der Realität zur Deckung gebracht werden.
